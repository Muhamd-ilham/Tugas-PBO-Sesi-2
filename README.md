1. Definisi Class dan ObjectClass: Sebuah blueprint atau kerangka dasar yang mendefinisikan atribut (data) dan metode (perilaku) yang akan dimiliki oleh objek.Object: Instansiasi atau wujud nyata dari sebuah class yang memiliki status (state) dan perilaku tertentu.2. Jenis-jenis MethodMethod Void: Method yang hanya melakukan aksi tanpa mengembalikan nilai (menggunakan keyword void).Method Non-Void (Return): Method yang mengembalikan nilai dengan tipe data tertentu (seperti String, int).Method Static: Method yang menempel pada class sehingga dapat dipanggil tanpa membuat objek (contoh: main method).3. Penjelasan Bagian Kode (Gambar Komputer)Berdasarkan gambar pada file tugas :Nama Class: Deklarasi class dengan nama Komputer.Atribut/Variabel: Data yang dimiliki objek (jenis_komputer dan merk).Method Setter: setDataKomputer untuk mengisi nilai ke dalam atribut.Method Getter (Jenis): Mengambil/mengembalikan nilai jenis_komputer.Method Getter (Merk): Mengambil/mengembalikan nilai merk.Instansiasi Objek: Membuat objek baru mykom dari class Komputer.Pemanggilan Method: Mengisi data ke objek menggunakan setDataKomputer.Output: Mencetak hasil pengembalian nilai getter ke layar monitor.
